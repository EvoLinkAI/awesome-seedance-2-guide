## 📋 Parametre Özellikleri

| Giriş Türü | Desteklenen Formatlar | Miktar Sınırı | Boyut Sınırı | Süre Sınırı |
|----------|----------|----------|----------|----------|
| Görüntü | jpeg, png, webp, bmp, tiff, gif | ≤ 9 | Tek < 30MB | — |
| Video | mp4, mov | ≤ 3 | Tek < 50MB | Toplam 2-15s |
| Ses | mp3, wav | ≤ 3 | Tek < 15MB | Toplam ≤ 15s |
| Metin | Doğal dil | — | — | — |

**Birleşik Sınır:** Toplam ≤ 12 dosya (görüntü + video + ses birleşimi)

**Çıkış Özellikleri:** Oluşturulan süre 4-15s serbestçe seçilebilir, video 720p'ye kadar, ses efektleri/arka plan müziği dahil

> **Uyum Notu:** Şu anda gerçekçi insan yüzleri içeren materyallerin yüklenmesini desteklemez. İllüstrasyon stili, AI tarafından oluşturulan sanal karakterler, hayvanlar, ürünler, sahneler vb. kullanılması önerilir.

## 🎮 Etkileşim Yöntemi

İstemde `@malzeme_adı` kullanarak her malzemenin rolünü belirtin. Yükleme sırası numaralandırma sırasıdır:

```
@görüntü1 ilk kare olarak, @video1 kamera dili referansı olarak, @ses1 arka plan müziği olarak
```

| Giriş Noktası | Kullanım Durumu |
|------|----------|
| **İlk/Son Kare** | Sadece ilk kareyi (veya ilk + son kareyi) + istemi yükleyin |
| **Hepsi Bir Arada Referans** | Çok modlu kombinasyon girişi (görüntü + video + ses + metin) |

**Yaygın Yazım Desenleri:**

```
# İlk kareyi belirtin
@görüntü1 sahnenin ilk karesi olarak kullanın, ...

# Sadece kamera hareketini referans alın, karakteri değil
@video1 tüm kamera hareketi efektlerini referans alın, ancak karakter görünümü @görüntü1 kullanın

# Hareketi ve kamera hareketini ayrı olarak referans alın
@video1 karakter hareketini referans alın, @video2 dairesel kamera hareketini referans alın

# Video uzatma (oluşturulan süre = yeni saniye, toplam süre değil)
@video1 5 saniye uzatın, [içerik açıklaması]

# Video ses efektlerini referans alın
Arka plan BGM @video1 ses efektlerini referans alır
```

## 💡 İleri Teknikler

**İstem Yazımı**

- Uzun videolar (10s+) zaman çizelgesi segmentasyonu kullanın: `0-3s: [açıklama] / 3-6s: [açıklama]`
- Hareket/duygu spesifik olmalı: ❌ `karakter üzgün` → ✅ `gözyaşları yanaklarından aşağı akıyor, ağız köşeleri hafifçe titriyor`
- Tek çekiş sonunda eklenmelidir: `Tüm boyunca kesme yapma, tek çekiş.`

**Anahtar Kelimeler Belirli Efektleri Tetikler**

| İstenen Efekt | Önerilen Yazım |
|------------|----------|
| Hitchcock Zoom | `ana karakter korku hissettiğinde Hitchcock zoom` |
| Dairesel Kamera | `robot kol çok açılı dairesel` |
| Hız Kademeli Artış | `roller coaster hızı giderek hızlanıyor` |
| Parçacık Efekti | `altın kum dağılıyor` / `parçacık dağılma efekti` |

**Çok Modlu Kombinasyon Stratejisi**

| Kontrol Etmek İstediğiniz Boyut | Kullanılacak Malzeme |
|----------------|------------|
| Karakter Görünümü | Görüntü (çok açılı) |
| Kamera Hareketi Yöntemi | Video (kamera dili referansı) |
| Hareket | Video (hareket referansı) |
| Ses Tonu/Ton | Video (diyalog içeren referans video) |
| Arka Plan Müziği Ritimleri | Video veya Ses |
| Sahne Stili | Görüntü (sahne referans görüntüsü) |

**Sık Sorulan Sorular**

- **Referans video kamera hareketini çoğaltmıyor?** → `@video1 tüm kamera hareketi efektlerini tamamen referans alın` ekleyin
- **Karakter görünümü tutarsız?** → Çok açılı görüntü yükleyin, isteme `karakter görünümünü @görüntü1 ile tamamen eşleştirin` ekleyin
- **Video uzatma dikişi doğal değil?** → Uzatma isteminin başında orijinal videonun son karesinin durumunu açıklayın

## 📝 İstem Şablonları

**Ürün 360 Derece Gösterimi**
```
@görüntü1 [ürün adı] ana konu olarak, kamera hareketi @video1 referansı, [yakın çekim bölümü]ne yakınlaş,
kamera döndükten sonra [ürün] tam görünümü ters göster, [ürün özelliği detayları] net görün,
çevre ortamı [atmosfer açıklaması]
```

**Reklam Karşılaştırması**
```
Bu bir [ürün] reklamıdır, @görüntü1 ilk kare ekran olarak, [karakter A] [durum A, örn: zarif],
kamera hızla sağa pan, @görüntü2 [karakter B][durum B, örn: berbat] çekin,
kamera sola pan [ürün] çekin, [ürün] @görüntü3 referansı, [ürün] [çalışma durumu].
```

**Video Uzatma Senaryosu**
```
[N]s
@video1 [ileri/geri] [N] saniye uzatın.
[0-X]s: [ekran açıklaması].
[X-Y]s: [ekran açıklaması].
[Y-N]s: [bitiş ekranı/altyazı].
```

**Tek Çekiş**
```
@görüntü1@görüntü2@görüntü3..., [bakış açısı]tek çekiş [hareket yöntemi] çekiş,
[hareket yörüngesi: A'dan B'den geçerek C'ye ulaş], [hız/ritim değişimi].
Tüm boyunca kesme yapma, tek çekiş.
```
