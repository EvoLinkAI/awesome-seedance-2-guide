## Case 2-3-8-1 · 顛覆劇情（古裝橋上推落）

**輸入：** 1個原視頻

### Prompt

```
顛覆@視頻1裡的劇情，男人眼神從溫柔瞬間轉為冰冷狠厲，在女主毫無防備的瞬間，猛地將
女主從橋上往外推，把女主推進水裡。動作乾脆利落，帶著蓄謀已久的決絕，沒有絲毫猶豫，
徹底顛覆原有的深情人物設定。女主墜入水中的瞬間，沒有尖叫，只有難以置信的眼神，她
抬頭衝男主嘶吼："你從一開始就在騙我！"男主站在橋上，臉上露出陰冷的笑容，對著水面
低聲說："這是你欠我家族的。"
```

## Case 2-3-8-2 · 顛覆劇情（酒吧零食廣告）

**輸入：** 1個原視頻

### Prompt

```
顛覆@視頻1的整個劇情
0–3秒：西裝男坐在酒吧，神情冷靜，手裡輕晃酒杯。環境音低沉，西裝男小聲說"這單生意，很大。"
3–6秒：身後的女人表情緊張問"有多大？"西裝男抬眼，壓低聲音："非常大。"
6–9秒：突然西裝男從桌下掏出—— 一大包體積誇張的零食禮包，"咚"的一聲重重放在桌上。
9–12秒：身後的女人肌肉從僵硬到鬆弛，整個人表情放鬆。
13–15秒：西裝男拿出一包零食給女人，字幕彈出"再忙，也要記得吃點零食~"
```

> **核心技巧：**
> - 顛覆劇情時按時間線寫清楚新情節（0-3秒/3-6秒...）
> - 角色替換加上"動作完全模仿原視頻"可保留原有動作
> - 局部修改說清楚保留什麼、改變什麼
