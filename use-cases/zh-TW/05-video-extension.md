## Case 2-3-5-1 · 驢騎摩托車腦洞廣告（延長 15s）

**輸入：** 2張角色參考圖 + 1個原視頻

### Prompt

```
延長15s視頻，參考@圖片1、@圖片2的驢騎摩托車的形象，補充一段腦洞廣告
畫面1：側面固定鏡頭，驢騎著摩托車衝出棚欄，旁邊的雞受到驚嚇
畫面2：驢騎著摩托在沙地盤旋，先特寫摩托輪胎，再切到半空中俯拍驢騎著摩托車做著盤旋特技，掀起煙霧
畫面3：背景是雪山鏡頭，驢騎著車從山坡飛越過，廣告語在主體背後，通過遮罩的形式
中間出現"Inspire Creativity,Enrich Life"，最後在摩托飛過，揚起一陣塵煙
```

```

11-15秒：英文漸顯第一行"Lucky Coffee"，第二行"Breakfast"，第三行"AM 7:00-10:00"。
```

## Case 2-3-5-4 · 街頭向日葵（向前延長 10s）

**輸入：** 1個原視頻

### Prompt

```
10s
向前延長10s，溫暖的午後光線裡，鏡頭先從街角那排被微風掀動的遮陽篷開始，慢慢下移到
牆根處幾株探出頭的小雛菊。緊接著，畫面裡出現主人公的紅色板鞋，他正蹲在街邊花攤前，
笑著把一大捧向日葵攏進懷裡，花瓣蹭過他的白T恤。他轉身踏上滑板時，花攤老闆笑著喊了
句"小心花瓣飛啦！"，他衝老闆揮了揮手，然後才開始滑行，幾片金黃的花瓣已經先一步從
花束裡掙脫出來，落在了滑板的板面。
```

> **核心技巧：**
> - 說明"向前延長"還是"向後延長"
> - 生成時長 = 新增的秒數（不是總時長）
> - 用時間節點分段（"1-5秒/6-10秒"）讓銜接更自然
