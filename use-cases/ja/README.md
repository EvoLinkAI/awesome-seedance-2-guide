# ユースケース概要

Seedance 2.0の10大コア機能をカバーする合計**55個のケース**。

| # | 機能 | ケース | 最適な用途 |
|---|------|:-----:|----------|
| [01](./01-consistency.md) | **一貫性の向上** | 6 | ショートドラマ、eコマース製品展示、固定キャラクター |
| [02](./02-camera-movement.md) | **カメラワークとアクション再現** | 7 | シネマティックスタイル、広告、ダンス/アクションシーン |
| [03](./03-creative-effects.md) | **クリエイティブテンプレート/複雑なエフェクト** | 8 | エフェクトイントロ、ブランド広告、スタイル転送 |
| [04](./04-story-completion.md) | **ストーリー補完** | 3 | マンガアニメーション化、絵コンテから動画へ |
| [05](./05-video-extension.md) | **ビデオ延長** | 4 | 既存ビデオの続編、広告エンディング |
| [06](./06-audio-voice.md) | **音声品質と真正性** | 10 | ダイアログビデオ、ボイスオーバー、多言語 |
| [07](./07-continuity.md) | **ワンショット連続性** | 5 | ドキュメンタリースタイル、トラッキングショット |
| [08](./08-video-editing.md) | **ビデオ編集** | 5 | プロット改変、キャラクター置換、製品配置 |
| [09](./09-music-sync.md) | **音楽同期** | 4 | ショートビデオ、MV、衣装チェンジ同期 |
| [10](./10-emotion.md) | **感情表現** | 3 | 広告比較、コメディ、感情的なショート |
